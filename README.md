# 🕊️ MemoNest – Digital Memory Journal

MemoNest is a modern web application designed to help users capture and preserve their daily memories in a simple and meaningful way.  
It combines a clean Apple-inspired interface with interactive features like a calendar view, rich text editing, and image uploads.

---

## ✨ Features

- 📅 **Interactive Calendar**
  - Navigate between months
  - View memories saved on specific dates
  - Visual indicators for days with entries

- 📝 **Rich Text Editor**
  - Format text (bold, italic, underline)
  - Create lists and headings
  - Align text (left, center, right)

- 🖼️ **Image Upload**
  - Attach an image to each memory
  - Preview images before saving
  - Display images inside calendar entries

- 🧠 **Memory Management**
  - Add, edit, and delete entries
  - View saved entries in a clean modal
  - Store data using browser localStorage

- 🎨 **Modern UI/UX**
  - Apple-inspired design
  - Smooth animations and transitions
  - Glassmorphism effects

---

## 🧱 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage (for data persistence)

---
## 🧪 Example Use Case

- Click on a day in the calendar  
- Add a title, note, and image  
- Save your memory  
- View it anytime by clicking the same date
